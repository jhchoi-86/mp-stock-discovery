## [cite_start]1. 프로그램 목적 및 비전 [cite: 92]
### 1-1. [cite_start]프로그램 목적 [cite: 93]
[cite_start]MP Stock은 국내외 주요 종목과 암호화폐를 대상으로 MP Stock이 자체 개발한 차트 분석 툴을 활용하여 다중 시간대별 기술적 분석을 수행하고, 그 결과를 점수화하여 매매 참고 정보(추천 종목·진입가·목표가·손절가)를 회원에게 제공하는 투자 참고 정보 프로그램입니다. [cite: 94]

| 핵심 원칙 | 본 프로그램은 회원이 직접 주문을 넣는 자동 매매 시스템이 아닙니다. [cite_start]모든 분석 결과는 당일 매매를 위한 참고 자료로만 활용되어야 하며, 이 원칙은 모든 설계·UI·법적 고지에 반드시 명시됩니다. [cite: 95] |

## [cite_start]4. 유료 회원 전용 기능 상세 [cite: 121]
### 4-1. [cite_start]진입가 도달 텔레그램 알람 [cite: 122]
[cite_start]유료 회원이 관심 종목으로 등록한 추천 종목의 현재가가 설정된 진입가에 도달할 경우, 텔레그램으로 실시간 알람을 발송합니다. [cite: 123]

| 항목 | 내용 |
| :--- | :--- |
| 알람 조건 | 현재가 ≤ 진입가 1 또는 진입가 2 또는 진입가 3 (회원이 선택한 진입가) |
| 알람 내용 | 종목명 / 현재가 / 도달한 진입가 번호 / 목표가 / 손절가 / 분석 점수 / 시간대 |
| 발송 채널 | 텔레그램 봇 (회원 개인 텔레그램 계정 연동 필수) |
| 중복 발송 방지 | 동일 종목·동일 진입가 기준 4시간 쿨다운 적용 (Redis TTL) |
| 알람 설정 | 회원 개인이 관심 종목·진입가 번호 선택 후 알람 ON/OFF 가능 |
| 발송 시간 | 시장 운영 시간 내 실시간 (국내: 09:00~15:30 / 미국: 한국시간 기준 / 코인: 24시간) |

### 4-2. TradingView 차트 연동 [cite: 127]
MP Stock이 자체 개발한 종목 차트 분석 지표를 TradingView 프로그램에서 직접 확인하고, 회원이 직접 TradingView에서 매매 판단에 활용할 수 있도록 정보를 제공합니다. [cite: 128]

| 항목 | 내용 |
| :--- | :--- |
| 제공 방식 | TradingView 차트 링크 + MP Stock 개발 Pine Script 지표 공유 |
| 지표 내용 | BB-MACD · RSI 동적 지지선 · 눌림목 시그널 · 진입가/목표가/손절가 라인 표시 |
| 접근 방법 | 유료 회원 로그인 후 추천 종목 상세 페이지에서 "TradingView에서 보기" 버튼 클릭 |
| 차트 권한 | TradingView 계정 연동 불필요. MP Stock 제공 공유 링크로 즉시 확인 가능 |
| 주의 사항 | TradingView는 외부 서비스이며, MP Stock은 차트 확인 정보만 제공. 실제 매매는 회원이 직접 수행 |
