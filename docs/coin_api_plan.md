# 💎 [PRD] 코인 AI 종목 발굴 및 통합 추천 시스템

## 1. 개요 (Overview)
현재의 주식 종목 발굴 시스템(Momentum, Scorring, AI Analysis)의 성공적인 로직을 **가상자산(코인) 시장**으로 확장합니다. 본 시스템의 목적은 단순한 가격 알림이 아닌, **급등 가능성이 높은 종목을 정밀하게 발굴하고 근거(AI 코맨트)를 포함하여 추천**하는 것에 있습니다.

## 2. 주요 기능 및 목표 (Core Features)

### A. 코인 종목 발굴 엔진 (Discovery Engine)
- **모멘텀 스캐닝**: 업비트/빗썸 전 종목을 대상으로 거래량 급증, 이평선 정배열, 골든크로스 종목을 실시간 포착.
- **정밀 스코어링**: 주식 종목 발굴과 동일하게 1H, 2H, 일봉 단위의 지표를 가공하여 0~100점 사이의 '발굴 점수' 부여.
- **눌림목 감지**: 급등 후 안정적인 기술적 반등 구간(눌림목)에 진입한 종목 우선순위 추천.

### B. AI 추천 근거 생성 (Insight Engine)
- **신뢰성 중심 뉴스 수집**: 일반 구글 뉴스 외에 **거래소 공식 공지사항(Upbit/Bithumb Announcement API/RSS)**을 최우선 순위로 수집하여 정보의 정확성 확보.
- **노이즈 필터링**: 소셜 미디어나 커뮤니티의 스팸성 호재를 배제하고, 실제 거래소 상장/이벤트/공시 데이터에 기반한 AI 코맨트 생성.
- **종목별 AI 코맨트**: "왜 이 코인을 추천하는가?"에 대한 기술적/기본적 분석 결과를 텔레그램 리포트에 포함.

### C. 추천 리포트 시스템 (Reporting)
- **통합 대시보드**: 주식 대시보드 내 '코인 추천' 탭을 신설하여 발굴된 종목 실시간 노출.
- **텔레그램 정기 리포트**: 9:00 AM 정기 브리핑 및 실시간 유망 종목 포착 시 '추천 리포트' 발송.

## 3. 기술 스택 및 연동 (Technical Stack)

- **Data Fetching**: 
  - **WebSocket Stability**: `Exponential Backoff` 재연결 로직을 적용하여 24/7 무중단 수집 보장.
  - **Normalization Layer**: 각 거래소의 상이한 티커 포맷(KRW-BTC, BTC_KRW 등)을 내부 통일 모델로 정규화하여 스코어링 일관성 유지.
- **AI Processing**: Python Gemini API 활용 (코인 특화 프롬프트 적용).
- **Database**: 로컬 Postgres를 활용하여 과거 추천 종목의 수익률 및 승률 추적.
- **Security**: 모든 API Key는 **조회 전용(Read-Only)**으로 제한하여 보안 리스크 원천 차단.

## 4. 주식 시스템과의 차별점 (Differentiation)
- **24/7 가동**: 주식과 달리 연중무휴인 코인 시장 특성에 맞춰 야간/주말 특화 알림 로직 적용.
- **변동성 필터**: 코인 특유의 극심한 변동성을 고려하여 '가짜 돌파(Fake-out)' 방지 필터링 로직 강화.
