# MP Stock 관리자 비밀번호 및 보안 관리 가이드

## 1. 개요
MP Stock 애플리케이션의 관리자 권한 및 소스코드 보안을 수호하기 위한 행동 지침 및 복구 가이드입니다. 

## 2. 관리자(ADMIN) 비밀번호 분실 시 복구 방법
현재 MP Stock의 관리자 비밀번호는 외부 데이터베이스가 아닌 서버 깊숙한 곳의 특수 환경 변수(`.env`) 파일에 오프라인 상태로 안전하게 격리되어 있습니다. 웹에서 비밀번호를 잊어버리셨더라도 마스터 열쇠(SSH 인증서: `mp-key.pem`)만 있으면 1분 안에 복구할 수 있습니다.

### [비밀번호 확인 및 변경 순서]
1. PC 터미널(PowerShell 등)에서 AWS 서버로 접속합니다.
   ```bash
   ssh -i "C:\Users\danbe\Documents\mp-key.pem" ubuntu@13.211.128.167
   ```
2. 에디터를 사용해 환경변수 설정 파일을 엽니다.
   ```bash
   nano ~/mp-stock-discovery/.env
   ```
3. 열린 파일 내용 중 `ADMIN_PASSWORD=...` 줄을 찾으시면 기존 비밀번호를 바로 확인할 수 있습니다.
4. 만약 비밀번호를 변경하고 싶으시다면 해당 줄의 비밀번호를 지우고 새 비밀번호를 적어 넣습니다.
5. 저장(`Ctrl+O` 후 `Enter`)하고 에디터를 종료(`Ctrl+X`)합니다.
6. 백엔드 서버를 재가동하면 즉시 새 비밀번호가 적용됩니다.
   ```bash
   pm2 restart mp-stock-discovery
   ```

## 3. 관리자 보안 권한의 안전한 보관 방법 (Best Practice)
현재 보안 체계에서 가장 탈취하기 어렵고 보호해야 할 **절대 권한 파일은 웹사이트 비밀번호가 아니라 서버에 직접 들어갈 수 있게 해주는 `mp-key.pem` 파일입니다.** 이 파일만 보유하면 언제든 시스템 통제 권한을 되찾을 수 있습니다.

### 🔑 `mp-key.pem` 파일의 절대 보안
현재 PC에 보관 중이신 이 파일은 개인 USB 메모리나 외장하드에 복사본을 만들어 금고 또는 물리적으로 안전한 오프라인 장소에 다중 보관하시는 것을 강력히 권장합니다.

### 💾 `.env` 환경변수 오프라인 백업
서버에 세팅된 텔레그램 토큰, KIS API 키, 관리자 비밀번호, 그리고 핵심 알고리즘 통합 해시가 담긴 `.env` 파일 내용 전체를 복사하세요. 로컬 PC의 안전한 메모장(예: `서버키백업_절대유출금지.txt` 등)에 오프라인 백업해 두십시오. 만에 하나 서버 인스턴스가 날아가더라도, 해당 백업 텍스트만 덮어씌우면 단 5분 만에 완벽한 서버 복구 및 재구축이 가능해집니다.

### 🛡️ 웹사이트 비밀번호 관리자 사용
1Password, Bitwarden 같은 검증된 비밀번호 관리 앱이나 크롬 브라우저의 전용 암호 관리 기능을 활용해 웹사이트 화면 접속 비밀번호 역시 암호화 저장해 유지하십시오.

**핵심 요약:** 웹사이트 암호를 분실하더라도 `.pem` 키 파일만 잘 보관 중이시라면, 서버 터미널의 `.env` 파일을 열어 언제든 시스템에 개입하고 가장 강력한 권한으로 암호를 복구할 수 있습니다.
