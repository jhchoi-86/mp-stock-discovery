# [v6.6.0] 작업계획서: 랜딩페이지 성과 분석 통합

본 계획서는 '종목 성과 분석' 대시보드를 일반 사용자가 접근 가능한 랜딩 페이지로 통합하고, 유무료 회원별 최적화된 UX를 제공하는 것을 목적으로 합니다.

## 주요 변경 사항

### 1. 백엔드 보안 API 확장
- **엔드포인트**: `/api/public/daily-snapshots`, `/api/public/daily-snapshot-dates`
- **보안**: JWT 기반 권한 체크를 통해 `PAID` 등급 이상의 사용자에게만 성과 원본 데이터 노출.

### 2. 프론트엔드 UI/UX 통합
- **탭 네비게이션**: `Home`, `MP 시그널`, `Daily 성과` 3개 탭을 헤더에 배치하여 직관적인 이동 지원.
- **반응형 디자인**: 
    - PC: 고해상도 성과 분석 테이블 유지.
    - Mobile: 복잡한 테이블 대신 카드 리스트(Card View) 형태로 성과 요약 표시.

## 검증 계획
- 유료/무료 계정별 접근 권한 테스트.
- PC 및 모바일 브라우저에서의 탭 전환 및 데이터 렌더링 정상 여부 확인.
