# 체인지로그 (Changelog)

이 문서는 MP 주식종목발굴 시스템의 최신 업데이트 내역을 버전별로 기록합니다.

## [v1.3.0] - 2026-03-18 (Phase 15)
### 추가 (Added)
- **종목별 당일 수급 현황 표시**: '전종목 자동 동기화' 실행 시, 종목 데이터 분석 카드 하단에 당일 누적 수급량이 자동으로 표시됩니다.
  - **거래대금**: 한국투자증권(KIS) API(`acml_tr_pbmn`)를 활용하여 누적 거래대금(백만 원) 데이터를 표시합니다.
  - **기관 및 외국인 순매수**: 네이버 모바일 금융 통합 검색 API를 백그라운드로 호출하여 각 종목별 외국인(`foreignerPureBuyQuant`), 기관(`organPureBuyQuant`) 순매수 수량을 가져오도록 구현했습니다.
- **수급 UI 스타일링**: 양수(순매수)는 붉은색, 음수(순매도)는 푸른색 텍스트로 자동 조정하여 직관성을 높였습니다. (`SignalIndicator.jsx`)
### 최적화 (Optimized)
- 웹 크롤링 대신 속도가 빠르고 밴(ban) 위험이 적은 숨겨진 모바일 JSON API 엔드포인트를 채용함으로써 다량의 종목 동기화 속도를 일정하게 유지했습니다. 데이터 누락 방지를 위한 타임아웃 세이프가드 처리가 적용되었습니다. (`server.cjs`)

## [v1.2.1] - 2026-03-18 (Phase 14)
### 변경 (Changed)
- **전종목 자동 동기화 버튼 접근 권한 변경**: 기존에 최고 관리자(ADMIN) 모드에서만 보이도록 제한되었던 '[전종목 자동 동기화]' 버튼과 '[초기화 리셋]' 버튼의 종속 관계를 분리했습니다.
- 이제 일반 로그인 회원도 자신의 대시보드 화면상에서 직접 전종목 동기화를 트리거할 수 있습니다. (`App.jsx`)

## [v1.2.0] - 2026-03-17 (Phase 13)
### 추가 및 변경 (Added & Changed)
- **프리미엄 SaaS 로그인 랜딩 화면 전면 리뉴얼**: 구형 로그인 폼을 제거하고 최신 다크 모드 + 글래스모피즘(Glassmorphism) 테마로 UI를 일신했습니다.
  - 모바일에 최적화된 6:4 스플릿 레이아웃 적용 (반응형).
  - 배경 이미지를 관통하는 투명 효과 블러(Blur) 폼 박스, 네온 광원 효과 테두리 및 그라데이션 버튼 추가.
  - 좌측 공간에 `lucide-react` 아이콘(Bot, Zap, Target)을 배치하여 핵심 기능 어필 문구 정리.
- 핵심 인증 및 Zustand 상태 관리, 에러 처리 등의 비즈니스 로직(JavaScript)은 기존과 100% 동일하게 유지되어 안정성을 보장합니다. (`Login.jsx`, `index.css`)
