# 데이-스윙 통합형(Hybrid) 매매 전략 배점 분석 보고서

두 가지 핵심 매매 스타일을 동시에 만족하는 "황금 지점(Golden Zone)"을 판별하기 위한 v3.3.0 배점 엔진 분석 결과입니다.

## 1. 전략적 중첩 구간: "Golden Zone"

데이 트레이딩과 스윙 트레이딩의 요구사항이 만나는 지점은 **속도(Speed)**와 **구조(Structure)**가 동시에 확인될 때입니다.

### 통합형 종목의 핵심 특징
- **데이 타겟**: 30M/1H의 즉각적인 거래량 폭발과 매수 신호 발생.
- **스윙 타겟**: 2H/4H의 안정적인 추세 유지와 정배열 지지층 형성.

---

## 2. 통합 만족을 위한 배점 구조 분석 (v3.3.0 기준)

통합형 투자자에게 가장 매력적인 점수대는 **70점~85점** 구간입니다.

| 배점 항목 | 성격 | 분석 |
| :--- | :--- | :--- |
| **기본 50점** (2H 추세/눌림목/정배열) | **스윙(Swing)** | 종목의 펀더멘털 추세가 "우상향"임을 보장. |
| **중첩 가점 10~15점** (HH/Trend Overlap) | **데이터(Day)** | 하위 분봉(30M/1H)에서 현재 실시간 수급이 들어오고 있음을 의미. |
| **강력/절대 보너스 4~8점** | **공통(Strong)** | 이동평균선과 볼린저밴드의 응축 후 발산(Squeeze) 시점으로 두 전략 모두에 최적의 진입 시점. |
| **거래량 가점 3점** | **공통(Vol)** | 유동성 공급의 증거. |

### 💡 전략적 판별식
- **70점 이상**: 스윙으로 가져갈 수 있는 추세적 안정성 위에, 장중 데이 트레이딩이 가능한 수급이 붙기 시작한 상태.
- **80점 이상**: 모든 타임프레임이 한 방향을 가리키는 '강력 홀딩' 구간.

---

## 3. 두 마리 토끼를 잡기 위한 배점 고도화 제언

만약 "데이/스윙 혼합형"으로 시스템을 더 정밀하게 사용하고 싶다면, 다음의 **2단계 필터링**을 추천합니다.

### 1단계: 점수 임계값 설정
- **Score >= 65점**: 하이브리드 진입 가능권.
- **Score >= 75점**: 데이-스윙 모두에 강력 추천.

### 2단계: 필수 조건 필터 (And Logic)
단순 합산이 아닌, 아래 두 가지가 **동시 만족**될 때 하이브리드 종목으로 자동 분류하는 기능이 효과적입니다.
1. `2H cond_up7` == True (**스윙 확신**)
2. `30M signal_HH` == True (**데이 타점**)
3. `1D trigger_vol` == True (**수급 확신**)

---

### 📝 레드팀 총평
v3.3.0 엔진은 **"스윙 기반의 안정성 위에 데이의 변동성 가점을 얹은 구조"**입니다. 따라서 점수 자체가 **통합형 지표**로서 매우 우수한 기능을 수행하고 있습니다. 

하이브리드 투자자라면 **대시보드에서 72점(동국제약 사례 등) 이상인 종목**을 타겟팅하되, **30M/1H의 신호가 (O)인 상태**에서 진입하면 두 전략의 이점을 모두 누릴 수 있습니다.
