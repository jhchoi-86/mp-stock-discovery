# MP Stock Platform - Release History

이 파일은 플랫폼의 코드 수정 및 배포 이력을 관리합니다.

## [v3.0.0] - 2026-04-01
### 🚩 상태: 전 유니버스(350종목) 7-TF 동기화 및 UI 고도화 완료

### 🛠 주요 변경 및 조치 사항
1.  **7개 타임프레임(30M~1W) 통합 분석 엔진 구축**
    -   기존 1H/1D 위주에서 30M, 4H, 2D, 1W 등 7개 TF 정밀 분석 로직(`analyzer.cjs`)으로 확장.
    -   **2D 타임프레임** 전용 리샘플링 및 신호 판정 로직 안정화.
    -   **범용 인터벌 캐시** 도입으로 1D/2D 분석 시 중단 없는 캐시 히트율(100%) 달성.
2.  **UI/UX v1.1 고도화 (PcDashboard / Mobile)**
    -   **동적 TF 필터**: 대시보드 상단 7개 TF 버튼을 통한 원클릭 종목 선별 기능 추가.
    -   **이평선 정렬 시각화**: 2H 이평선(5/10/20/60) 가격순 정렬 및 현재가(📍) 위치 강조 렌더링.
3.  **데이터 무결성 확보**
    -   코스피 200 / 코스닥 150 누락 종목(효성, HLB제약 등) 추가 및 전수 동기화(2,450개 신호).
    -   Atomic Write 방식을 통한 `signals.json` 파일 손상 방지.

## [v1.0.1] - 2026-04-01
### 🚩 상태: 운영 서버(AWS) 배포 완료

### 🛠 주요 변경 및 조치 사항
1.  **플랫폼 긴급 복구 (Rollback)**
    -   3월 31일 오전 1시 14분에 생성된 **v1.0.0 전체 백업 파일**을 기반으로 서버 및 로컬 코드 완전 복구.
    -   실험적 기능 추가 과정에서 발생한 UI 불일치 및 모바일 크래시 문제 해결.
2.  **통합 자동 동기화 최적화 (Sync Patch)**
    -   **프론트엔드**: 대량 데이터(350종목) 처리 시 발생하는 연결 끊김 방지를 위해 Axios 타임아웃을 **5분 → 15분으로 연장**.
    -   **백엔드**: 동기화 처리 속도 향상을 위해 종목별 대기 시간을 **150ms → 100ms로 단축**.
3.  **모바일 안정성 확보**
    -   모바일 대시보드의 `Virtuoso` 및 `lucide-react` 아이콘 참조 오류(`ReferenceError`) 수정 완료.
